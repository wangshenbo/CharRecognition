一个轻量级的验证码识别程序

开发语言：C++
支持库：OpenCV

原理很简单：首先进行前景提取，采用OSTU算法求域值，再对图像进行二值化，去除一些孤立的噪点，然后提取连通域，对每个连通域进行合并，合并要求是两个连通域之间的距离：x方向和y方向不超过3个像素，这样第个连通域就包括一个单独的字符，按所得到的连通域提取每个字符，并进行归一化成16*16大小的图像，将图像的每行组合成一个256大小的输入向量，然后通过神精网络算法对样本集进行训练与预测，得到最终的验证码。

说明：TrainML.cpp是用于训练神精网络的，CCharRecogintion类可用于收集样本与识别验证码，具体用法参考main.cpp
